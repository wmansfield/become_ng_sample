Solid Color Menu Background

This table contains all the solid color menu background that looks great when filled in the menu sidebar background. They happens to be the app colors as well. You can always check our demo and change solid colors from the customization panel found on the right side.

All these options can be set changing data-background-color attribute :

Solid Colors Description
black You can fill black color in the menu background by changing data-background-color attribute's value to black found in the src/app/layouts/full/full-layout.component.html file.
white You can fill white color in the menu background by changing data-background-color attribute's value to white found in the src/app/layouts/full/full-layout.component.html file.
primary You can fill primary color in the menu background by changing data-background-color attribute's value to primary found in the src/app/layouts/full/full-layout.component.html file.
info You can fill info color in the menu background by changing data-background-color attribute's value to info found in the src/app/layouts/full/full-layout.component.html file.
success You can fill success color in the menu background by changing data-background-color attribute's value to success found in the src/app/layouts/full/full-layout.component.html file.
warning You can fill warning color in the menu background by changing data-background-color attribute's value to warning found in the src/app/layouts/full/full-layout.component.html file.
danger You can fill danger color in the menu background by changing data-background-color attribute's value to danger found in the src/app/layouts/full/full-layout.component.html file.

Menu Bg Image

How to change sidebar menu background image?

Change the data-image attribute's value found in the src/app/layouts/full/full-layout.component.html file to whatever image you want to display as sidebar menu background image.

Where to put background image?

Many sidebar background image options can be found in the src/app/assets/img/sidebar-bg/ folder.

Recommended image size?

Recommended image size for the sidebar menu background image is 400 X 800.

Compact Menu

If you are one of those like me who loves menu to be compact by default then you need not worry. Just add below two class with wrapper class found in the src/app/layouts/full/full-layout.component.html file.

nav-collapsed menu-collapsed

Menu Width

You have 3 different menu sidebar sizes to choose from.

All options are defined in below table :

Options Classes Description
Small sidebar-sm Add sidebar-sm class to the .wrapper class found in the src/app/layouts/full/full-layout.component.html file.
medium No class required By default the menu size is medium and you do not need to add or remove any class.
large sidebar-lg Add sidebar-lg class to the .wrapper class found in the src/app/layouts/full/full-layout.component.html file.

Fixed Top Navbar

If you want top navbar to be fixed, then it's easy now. Just uncomment one line which says @import "core/fixed-navbar" found in the \src\assets\sass\scss\_core.scss file. And That's it!
