Brown Background Color
Background Color Options Class Description
Background Color .bg-brown Brown Class for background color.
Lighten Color .bg-brown.bg-lighten-* For lighten brown background color, this two classes needed. Here *: 1,2,3,4 lighten brown color options.
Darken Color .bg-brown.bg-darken-* For darken brown background color, this two classes needed. Here *: 1,2,3,4 darken brown color options.
